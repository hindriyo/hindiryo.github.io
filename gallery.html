Galeri Hobi

Bermain Gitar

Saya sangat suka bermain musik dan salah satu alat musik saya dapat saya mainkan adalah gitar.

Bermain Basket

Apabila sedang luang saya suka bermain basket bersama teman-teman di lapangan.

Ternak Lele

Saya memiliki kolam ikan lele di belakang rumah, jadi setiap pagi hari saya memberi pakan ikan lele tersebut.

Ternak Burung Dara

Di atap rumah saya ada kandang burung dara. Setiap sore saya membersihkan kandang burung dara.

Fotografi

Saya suka mengabadikan momen momen indah dan pesona langit yang begitu memukau.

Lari Pagi

Udara di pagi hari sangat segar. Jadi saya saat hari minggu memutuskan untuk lari pagi keliling kampung.

Memasak

Karena ibu dan ayah saya pergi bekerja, jadi saya diharuskan untuk bisa memasak untuk kebutuhan sehari hari.

Berenang

Saat libur sekolah, saya dan adik saya sering berenang. Karena berenang memiliki banyak manfaat, salah satunya adalah meredakan stres

Main Game

Setelah jenuh karena belajar seharian, biasanya saya bermain game. Saya senang bermain game karena mendapatkan kesenangan saat menang di game tersebut

Nonton Anime

Selain bermain game saya juga menyukai jejepangan. Salah satunya adalah kartun jepang karena ceritanya menarik, grafisnya sangat bagus, dan sangat memanjakan mata.
